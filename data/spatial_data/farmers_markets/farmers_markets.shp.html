Process_Step:
Process_Description:
Data table dervied from Puget Sound Fresh website was imported to GIS layer. Data was manipulated to conform to COMMON_INTEREST core schema. Items not part of core schema were added to a business table for this topic. Data was imported to respective core and business tables. Featureclass FARMERS_MARKETS_POINT was generated using standard python routine to create children featureclasses from COMMON_INTEREST and associated business tables.
Process_Date: 201102
Process_Step:
Process_Description:
Updated against list provided by Puget Sound Fresh website. Spreadsheet contained 74 records, only 52 of these records were updated to the FARMERS_MARKET_POINT featureclass. Rejected records include those whose geographic location falls outside bounds set for King County area of interest. Records from PSF spreadsheet reformatted to align to schema for COMMON_INTEREST, but all original data values preserved. Original data in COMMON_INTEREST was truncated for this domain and reloaded; therefore FEATURE_ID values are not preserved during reload.
